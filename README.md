# Daily Expense Analytics Dashboard

## Project Overview

The Daily Expense Analytics Dashboard is a web-based application developed using React.js that helps users manage their daily income and expenses efficiently. The application provides a simple and user-friendly interface for tracking financial transactions, calculating total balance, and analyzing spending patterns through category-wise summaries.

This project is designed for working professionals, students, and individuals who want to monitor their financial activities and gain insights into their spending habits.

---

## Objectives

* Track daily income and expenses.
* Categorize transactions for better financial management.
* Calculate total income, total expenses, and available balance.
* Provide category-wise expense analytics.
* Improve personal budgeting and spending awareness.

---

## Features

### 1. Add Income and Expense

Users can add financial transactions by selecting the transaction type, entering a category, and specifying the amount.

### 2. Category-Wise Tracking

Transactions are organized into categories such as:

* Salary
* Food
* Transport
* Shopping
* Bills
* Entertainment
* Healthcare
* Education
* Travel
* Other Expenses

### 3. Total Balance Calculation

The system automatically calculates:

* Total Income
* Total Expenses
* Current Balance

### 4. Expense Analytics

Displays a category-wise summary of expenses, helping users identify where most of their money is being spent.

### 5. Responsive Dashboard

The application is designed with a clean and responsive user interface that works across desktop and mobile devices.

---

## System Workflow

1. User enters transaction details.
2. Transaction is stored in the application state.
3. Income and expense totals are calculated automatically.
4. Available balance is updated in real time.
5. Expense data is grouped by category for analytics.
6. Dashboard displays financial summaries and transaction history.
